Hybrid Energy Storage System (HESS) for Electric Go-Kart

The project involved development of a Hybrid Energy Storage System (HESS) for an electric go-kart. I designed and implemented a system that combines a battery and a supercapacitor to optimize energy efficiency, improve performance, and extend battery life.


Key Achievements:

  • Implemented a dynamic energy management system that allocates power demands optimally between battery and supercapacitor
  • Designed regenerative braking capability, capturing up to 30% of braking energy
  • Created robust safety diagnostics with real-time monitoring and fault detection
  • Developed a scalable solution that can be adapted for various electric vehicle applications

Technologies Used:

  • STM32 microcontroller for system control and management
  • Advanced PCB design for power electronics interface
  • Custom battery management system (BMS)
  • Energy modeling and simulation in MATLAB SIMULINK

Optimizing Energy Distribution in Hybrid Power Systems

This project focused on analyzing and optimizing hybrid power systems combining photovoltaic (PV), battery, and diesel generator setups. It involved parameter measurement, simulation, and results interpretation to improve system efficiency and minimize operational costs.


Key Contributions:

  • Leveraged Maximum Power Point Tracking (MPPT) for PV optimization
  • Implemented battery models to manage charge and discharge cycles efficiently
  • Developed economic dispatch algorithms that reduced diesel fuel consumption by 22%
  • Created energy flow management systems that improved overall system efficiency by 18%

Results:

The findings demonstrated significant cost savings, enhanced power flow management, and the potential for sustainable energy integration in various applications ranging from microgrids to commercial power systems.

Motor Driver and Power System for Maze Navigation Robot

This project involved independent design of a Motor driver and Power subsystem for a Maze Navigating robot. I developed a low-power H-bridge motor driver from first principles using a combination of N & P channel Mosfets.


Technical Highlights:

  • Designed transistor-based gate drivers for optimal switching performance
  • Developed firmware using the Dual-core Tensilica Xtensa LX6 microprocessor
  • Implemented I2C communication protocol for reliable subsystem integration
  • Created power management solutions that extended battery life by 35%
  • Conducted extensive SPICE simulations to validate design before implementation

Challenges Overcome:

Successfully addressed thermal management issues and switching losses through innovative circuit design and component selection, resulting in a highly efficient system that outperformed commercial alternatives.

Data Analysis on 911 Emergency Calls

Conducted a comprehensive data analysis project on 911 emergency call data to uncover trends, patterns, and insights. The project involved data cleaning, visualization, and statistical analysis.


Core Development:

  • Performed data cleaning and preprocessing to handle missing values and inconsistencies
  • Utilized Python libraries such as Pandas, NumPy, and Matplotlib for data manipulation and visualization
  • Conducted exploratory data analysis (EDA) to identify key trends and patterns
  • Developed interactive visualizations using Plotly and Dash for better data interpretation

Results:

The analysis provided valuable insights into the frequency and distribution of emergency calls, peak times for different types of emergencies, and geographical hotspots. The findings can help improve emergency response strategies and resource allocation.
